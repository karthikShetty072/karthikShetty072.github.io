The Library of the Governemnt First Grade College, Koppa was established since its inception. The Library aims to develop a comprehensive collection of documents that is useful for the faculty students and research scholars in their education and research activities the library has a collection of 7000 books that include the publication Archives ti the present future users and periodicals which include journals magazines. Library has also fully automated with easy libo-E library management software package with all the modules for use in the library. Computers are installed in the library mainly to access the OPAC of the library include the internet facility.
